## docs について

この`docs/`は、プロダクト「rikai（学習カリキュラム作成をAIが支援）」の仕様・設計・運用に関するドキュメント置き場です。
現状の実装はスケルトン段階のため、本ドキュメントは**これから実装していくべき到達点**を中心に記載しています。

## 目次

- **プロダクト概要**: `product_overview.md`
- **要件定義（機能/非機能）**: `requirements.md`
- **ユーザーフロー**: `user_flows.md`
- **情報設計・データモデル案**: `data_model.md`
- **API設計案（v1）**: `api.md`
- **アーキテクチャ（現状/目標）**: `architecture.md`
- **ロードマップ**: `roadmap.md`

## このリポジトリの現状（簡易）

- **frontend**: Bun + React。現状は`/api/hello`を叩く簡易APIテスターを含むサンプル実装。
- **backend**: Bun + Hono。現状は`GET /`のみのサンプル実装。

以降の設計では、将来的にフロントからバックエンド（`backend`）のAPIを呼び出して、
ユーザーの目標・制約条件・進捗に応じてカリキュラムを生成/更新する前提で記載します。
